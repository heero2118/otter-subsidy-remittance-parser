# Otter Learning — Remittance Parser

Parses SC ABC, KY CCAP, CAPS, PFCC, and FL VPK SR remittance files into a unified table with CSV export.

## Setup

```bash
npm install
npm run dev
```

Then open http://localhost:5173

## Deploy to StackBlitz (recommended for sharing)
1. Go to https://stackblitz.com/fork/github
2. Or drag this folder into https://stackblitz.com

## File Types
| Agency | Format |
|--------|--------|
| SC ABC | .pdf |
| KY CCAP | .pdf |
| CAPS | .csv |
| PFCC | .csv |
| FL VPK SR | .xlsx |
